## Work Done

### Masking Curriculum Implementation
Implemented the PO-GRL-style masking curriculum across three layers:

1. **Wrapper layer** (`envs/wrappers.py`):
   - Added `p_full_obs` parameter to `SingleAgentPEWrapper.__init__`
   - Added per-timestep observation override in `step()`: with probability `p_full_obs`, rebuild the observation with `los_blocked=False` (full visibility)
   - Uses `np.random.default_rng()` for independent random state

2. **Environment creation** (`training/amsdrl.py`):
   - Added `p_full_obs` parameter to `_make_partial_obs_env()` and `_make_vec_env()`
   - Passed through to `SingleAgentPEWrapper` on construction

3. **Training loop** (`training/amsdrl.py`):
   - Added `masking_curriculum`, `p_full_anneal_steps`, `p_full_residual` params to `AMSDRLSelfPlay.__init__`
   - In `_run_micro_phases()`: linear annealing of `p_full_obs` from 1.0 → `p_full_residual` over `p_full_anneal_steps`
   - Updates `p_full_obs` on all pursuer sub-envs via wrapper chain traversal before each micro-phase
   - Logs `p_full_obs` in eval entries and status line

4. **CLI** (`scripts/train_amsdrl.py`):
   - Added `--masking_curriculum`, `--p_full_anneal_steps`, `--p_full_residual` flags

5. **Tests** (`tests/test_env.py`):
   - `test_masking_curriculum_p_full_obs`: Verifies p_full_obs=1.0 always gives full obs
   - `test_masking_curriculum_p_full_obs_zero`: Verifies p_full_obs=0.0 preserves masking

All 679 non-BarrierNet tests pass.

## Decisions Made
| Decision | Rationale |
|----------|-----------|
| Only apply masking curriculum to pursuer envs | Pursuer is the one struggling with search under partial obs; evader doesn't need full obs override |
| Linear anneal formula: `p = (1 - progress) * (1 - residual) + residual` | Simple, smooth transition. Residual allows optional small full-obs probability after annealing |
| Per-timestep random override (not per-episode) | Matches PO-GRL approach. Each step independently decides whether to give full obs |
