## Decisions Made
| Decision | Rationale |
|----------|-----------|
| Radius-only masking when `sensing_radius` set (no LOS) | Different mechanisms for different experiments — combining would confuse results |
| Same radius for both agents (unless asymmetric) | Simpler; asymmetric_obs flag handles pursuer-only masking |
| Best-model checkpointing saves `evader_best.zip` | Prevents regression problem seen in S1v4a (-60pp) and S1v4b (-23pp) |
