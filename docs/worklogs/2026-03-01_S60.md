### 4. Training Results — S1v4a vs S1v4b (5M steps each, ~62 min)

**S1v4a (asymmetric LOS — pursuer masked, evader full obs + randomized 1-2 obstacles):**
| Phase | Escape Rate | Notes |
|-------|------------|-------|
| 0-1.3M | 1-12% | Flat exploration |
| 1.5M | 25% | Takeoff |
| 2.0M | 45% | Rapid climb |
| 2.45M | 73% | Continuing rise |
| **3.15M** | **80%** | **Peak** |
| 3.5-4.0M | 56-65% | Regression begins |
| 4.5-5.0M | 20-47% | **Severe late regression** |
| **Final** | **20%** | Regressed from 80% peak |

**S1v4b (symmetric LOS — both masked + randomized 1-2 obstacles):**
| Phase | Escape Rate | Notes |
|-------|------------|-------|
| 0-1.3M | 0-17% | Slow rise |
| 1.5M | 38% | Takeoff |
| 1.95M | 73% | Peak spike |
| 2.0-2.8M | 53-70% | Volatile plateau |
| 3.5-4.0M | 52-60% | Stable |
| 4.5-5.0M | 43-59% | Mild regression |
| **Final** | **50%** | More stable than S1v4a |

**Comparison summary:**
| Metric | S1v4a (asymmetric) | S1v4b (symmetric) | S1v3 (sym, fixed 2 obs) | S1v2b (full obs) |
|--------|-------------------|-------------------|------------------------|------------------|
| Peak escape | **80%** | 73% | 56% | **98%** |
| Peak step | 3.15M | 1.95M | ~5M (plateau) | 1.8M |
| Final (5M) | 20% | **50%** | 56% | **98%** |
| Late regression | **Severe (-60pp)** | Mild (-23pp) | None (plateau) | None |

**Key findings:**
1. **Asymmetric obs achieves highest partial-obs peak (80%)** but suffers catastrophic late regression
2. **Symmetric + randomized (S1v4b) is more stable** than S1v4a and slightly better than S1v3 (50% vs 56% final, but 73% vs 56% peak)
3. **Randomized obstacles helped both** exceed S1v3's 56% ceiling at peak
4. **Late regression is the dominant problem** — without checkpointing/early stopping, the final model underperforms the peak model
5. **Neither approaches S1v2b's 98%** — partial obs fundamentally makes the problem harder for the evader (losing opponent info hurts more than it helps vs a greedy pursuer that doesn't use info strategically)
