### Staleness-Based Search Reward
- Created `envs/search_staleness.py` with `SearchStalenessTracker` class
  - Discretizes arena into 10x10 grid cells
  - Tracks time since each cell was last observed by pursuer
  - Rewards pursuer for visiting stale (not recently observed) cells
  - Uses sensing_radius + LOS filtering for visibility
- Modified `envs/pursuit_evasion_env.py` to integrate tracker
  - Added `w_search` and `t_stale` params to `__init__`
  - Reset tracker on `reset()`
  - Compute search reward in `step()`
- Threaded params through `training/amsdrl.py` (`_make_pe_env`, `AMSDRLSelfPlay.__init__`, `env_kwargs`, config print)
- Added `--w_search` and `--t_stale` CLI flags to `scripts/train_amsdrl.py`
