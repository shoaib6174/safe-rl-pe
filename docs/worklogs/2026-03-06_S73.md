### Systematic Diagnosis

**Hypothesis 1: Code changes broke something**
Compared new code (f764a47) vs old code (1b05a47, pre-search-reward). The new code adds `w_search`, `t_stale`, `w_vis_pursuer` to env_kwargs, but all are properly gated with `if > 0` checks. When set to 0 (default), they have zero effect.

**Hypothesis 2: Warm-seeding without replay buffer causes catastrophic forgetting**
SAC warm-seeding loads the model but NOT the replay buffer (too large). The Q-network starts training on a narrow data distribution → confident-but-wrong value estimates. This could explain SP11e2's poor start despite good initial weights.

**Hypothesis 3: Missing collapse rollback / PFSP**
Checked SP11e2 launch logs — confirmed `--collapse_threshold 0.1 --collapse_streak_limit 5 --pfsp` were all correctly set.

**Hypothesis 4: Search reward diluting capture signal (SP11g2/h2)**
At `w_search=0.003`, the per-step search reward (~0.0015 mean) is ~50% of the distance reward magnitude (~0.003). This creates a competing dense signal. Reduced to `w_search=0.0003` for later runs.

### Code Bisect — Definitive Test
Launched two identical cold-start runs:
- **SP11i**: New code (f764a47), SP11e config, seed=42
- **SP11j**: Old code (1b05a47), SP11e config, seed=42

Results after 6 eval points were **BIT-FOR-BIT IDENTICAL**:
```
SP11i: 0.29, 0.39, 0.29, 0.40, 0.33, 0.32
SP11j: 0.29, 0.39, 0.29, 0.40, 0.33, 0.32
```

**Conclusion: Code changes have ZERO side effects when new features are disabled.**

### Root Cause: CUDA Non-Determinism + Seed Sensitivity
The true cause is that SP11e's 85% peak was a **lucky trajectory**. Without `torch.use_deterministic_algorithms(True)`, CUDA operations (atomicAdd in reductions, cuBLAS workspace selection, cuDNN algorithm autotuning) produce non-deterministic FP accumulation. Same seed + same code + different GPU state = different training trajectory.

This project already documented massive seed sensitivity (lesson #3 in MEMORY.md): same config produces capture rates 0.13 to 1.00. SP11e was one fortunate sample from a wide distribution.
